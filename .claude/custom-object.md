# Department
部署オブジェクト

## フィールド

| フィールド名 | 表示名 | 型 | 必須 | ユニーク制約 |
|---|---|---|---|
| id | 部署ID | テキスト | ✓ | あり |
| name | 部署名 | テキスト | ✓ | なし |
